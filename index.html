Para venda por PIX, cadastre o preço e a Chave PIX. A chave padrão já está preenchida: b8354404-5878-4776-9427-9407d1bd6292. O cliente verá um QR Code PIX válido e poderá copiar a chave. Para liberação automática real após confirmação, use a tabela pedidos_pix no Supabase ou um webhook do banco/PIX atualizando o status para aprovado. Para teste, use a URL com ?pago=1 após gerar o pedido. Para o upload funcionar, mantenha no Supabase Storage um bucket chamado catalogos.
